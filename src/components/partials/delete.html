To get a better understanding of how this boilerplate works, please check out its Readme file , and the original template detailed documentation It is also recommended to go through the docs for Webpack and vue-loader. If you have any issues with the setup, please file an issue at this boilerplate's repository.

Several external addins or libraries are implemented in this rig. You can checkout their pages to find documentation and related stuff :

  • Vue Resource : Http and Ajax requests from your Vue app.
  • Vue Router : Routing without page reloading.
  • VueX : State and event store management.
  • Vue Translate : Internationalisation plugin.
  • Bootstrap : SASS version of the styling framework, including components.
  • Font Awesome : SASS version of the icon provider.
